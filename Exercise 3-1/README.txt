Задание:

Напишите функцию sum/1, которая по данному положительному целому числу N возвращает сумму всех чисел от 1 до N.
Напишите функцию sum/2, которая по двум целым числам N и M, при N =< M, возвращает сумму всех чисел из интервала от N до M, включая концы, а при N > M программа останавливается с ошибкой времени выполнения.
