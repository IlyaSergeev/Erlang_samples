Задание:

Напишите функцию filter/2, которая принимает на вход список чисел и одно целое число, а возвращает список всех элементов списка, которые меньше либо равны числу, переданному вторым аргументом.
Напишите функцию reverse/1, которая переворачивает список.
Напишите функцию concatenate/1, которая преобразует список списков в один список, соединенняя все списки-элементы. Вам придется написать вспомогательную функцию и выполнить объединение списка в несколько действий.
Напишите функцию flatten/1, которая по списку вложенных списков строит линейный список.
