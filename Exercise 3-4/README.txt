Задание:

Напишите модуль, создающий дазу данных в которой можно хранить, записывать и удалять элементы. Функция destroy/1 удаляет базу данных. Сборщик мусора выполнит всю работу за вас. Но в том случае, если база данных хранится в файле, при вызове функции destory вам придется удалить этот файл. Функция destroy/1 включена для полноты интерфейса. При выполнении этого функциями из модуля lists пользоваться нельзя. Все рекурсивные функции должны быть написаны вами.
Используйте списки и кортежи в качестве основного типа данных.
